# StayFinder Hotel Search

A clean, responsive hotel listing and details website built with React, Vite, React Router, and the provided Demo Hotels API.

## Features

- Fetches live hotel data from `https://demohotelsapi.pythonanywhere.com/hotels/`
- Hotel details page powered by `https://demohotelsapi.pythonanywhere.com/hotels/:id/`
- Loading skeletons, error states, and empty result handling
- Search by hotel name or location
- Filter by city, rating, and price range
- Sort by price and rating
- Responsive cards, hero section, gallery, header, and footer
- Lightweight favorite button UI stored locally in the browser

## Tech Stack

- React
- Vite
- React Router
- CSS

## Getting Started

```bash
npm install
npm run dev
```

Open the local URL shown by Vite in your browser.

## API Endpoints

- Hotel listing: `https://demohotelsapi.pythonanywhere.com/hotels/`
- Hotel details: `https://demohotelsapi.pythonanywhere.com/hotels/:id/`
